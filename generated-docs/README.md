# Generated Documents

This folder stores generated project documents and local validation artifacts that are not part of the approved product/spec source of truth.

Use this folder for future agent-generated documents, including:

- `plans/`: implementation plans and task plans created during development.
- `handoffs/`: issue handoff notes for follow-up conversations.
- `logs/`: local validation or acceptance logs.

Keep approved SightSync V2 specifications in `docs/superpowers/specs/`. Those specs remain the source of truth referenced by `AGENTS.md`.

Do not commit raw device logs unless the user explicitly asks for them to be uploaded. Logs may contain personal utterances, package names, or local device details.
